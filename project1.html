AUTOMATED PLANT WATERING SYSTEM

Project Overview

This project is an IoT-based automated plant watering system that uses an Arduino, soil moisture sensor, and water pump to monitor soil conditions and water plants automatically. It also integrates the Blynk app for real-time monitoring and remote control.

GitHub Repository

Key Features

  • Automatically waters plants when soil is dry
  • Stops watering when soil is sufficiently wet
  • Manual pump control via Blynk app
  • Real-time moisture and system status monitoring
  • LED and buzzer alerts for system feedback
  • Mobile notifications for watering events

Tools & Components Used

Hardware

  • Arduino Uno Rev 2
  • Capacitive Soil Moisture Sensor
  • 5V Water Pump (externally powered)
  • Relay Module
  • LED + resistor
  • Buzzer
  • Breadboard + jumper wires
  • 5V USB Power Bank

Software

  • Arduino IDE
  • Blynk IoT App

How It Works

The soil moisture sensor reads moisture levels in the soil. The Arduino processes the data and determines whether the soil is dry or wet.

If the soil is dry, the relay activates the water pump. If the soil is wet, the pump is turned off.

Status updates are sent to the Blynk app in real time.

Challenges & Solutions

Pump required more power than Arduino could provide
Solution: Used an external 5V USB power source for safe operation.

Sensor readings were unstable due to wiring issues
Solution: Rewired using jumper wires for stable connections.

Pump would not stop correctly
Solution: Added threshold buffering (hysteresis) in code.

Wi-Fi and Arduino control overload
Solution: Split system — Arduino handles hardware, ESP/D1 Mini handles Wi-Fi.

Voltage mismatch between Arduino and ESP
Solution: Used a voltage divider to protect components.

Serial communication issues
Solution: Adjusted baud rates and fixed wiring for stable sync.
